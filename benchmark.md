# Benchmarking

1. If there is an issue with permission run `chmod +x benchmark.sh`
2. Build `pmo.c` before running
3. Run `./benchmark.sh output_file no_of_runs`  
   - e.g. ./benchmark.sh benchmark_blocked.txt 10
